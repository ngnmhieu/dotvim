# Terminal setting in macOS

**Link:** [Dracula Theme](https://draculatheme.com/terminal)

**Clone Git Repo**
```
$ git clone https://github.com/dracula/terminal.app.git
```

**Activate the theme**

1. Terminal > Settings Tab
2. Click "Gear" icon
3. Click Import...
4. Select the Dracula.terminal file
5. Click Default
